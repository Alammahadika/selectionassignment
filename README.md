## Assignment Summary & Data

This assignment focuses on analyzing data from **100 universities from various countries**. The data provided includes several key UI GreenMetric indicators, represented as scores:

* **Score EC**: **Economic Contribution / Impact** â€“ The university's contribution to the economy, such as graduates entering the workforce, startups, patents, etc. In this analysis, the EC Score is also interpreted as a reflection of the **Social Impact** generated by the university through its economic contributions and innovation.
* **Score WS**: **World Scientific / Web of Science** â€“ Scientific publications in internationally reputable journals or citation score.
* **Score WR**: **World Ranking** â€“ The university's relative position in global rankings (e.g., QS, THE, Shanghai Ranking).
* **Score TR**: **Teaching & Research** â€“ The quality of teaching and research at the university (e.g., number of PhD faculty members, research funding).
* **Score ED**: **Education Delivery / Educational Development** â€“ Curriculum quality, learning facilities, accessibility, and academic development.
* **Score SI**: **Social Impact** â€” The university's social contributions to society, such as community service activities, local collaborations, inclusiveness, the social impact of research, and its role in sustainable development.

The main objective of this assignment is to analyze this data to understand the **competitive position and academic strength** of the universities.

## Mapping University Competitive Positions

```r
library(ggplot2)
 library(dplyr)
 library(tidyr)
 library(forcats)
 library(ggrepel)
 

 data <- data_univ %>%
   mutate(
     Competitive_Edge = (`Score EC` + `Score WS`) / 2,  # Daya saing
     Academic_Strength = (`Score TR` + `Score ED`) / 2   # Kekuatan akademik
   )

 data <- data %>%
   mutate(
     Competitive_Edge_Scaled = Competitive_Edge / 100,
     Academic_Strength_Scaled = Academic_Strength / 100
   )

 ggplot(data = data, aes(x = Competitive_Edge_Scaled, y = Academic_Strength_Scaled)) +
   # Menggunakan Competitive_Edge_Scaled dan Academic_Strength_Scaled
   geom_point(aes(color = Region), size = 1.5, alpha = 0.7) +
   
   geom_text_repel(aes(label = `University Name`), size = 3) +
   
   labs(title = "Competitive Positioning of Universities",
        x = "Competitive Edge (Employability & Reputation)",
        y = "Academic Strength (Teaching & Education)") +
   

   scale_color_discrete(name = NULL) +
   
   scale_x_continuous(breaks = c(5, 7, 10, 12, 15, 18), # Sesuaikan dengan rentang data setelah dibagi 100
```
